### 1.3 Mathematical Code Integrity Verification
1. **`scripts/proyeccion_20d.py`**:
   - Optuna WFO: Uses `TPESampler(seed=42)` for reproducible optimization (line 94). Runs 150 trials per 6h window.
   - Indicator calculations: Computes ATR(14), EMA(20), RSI(14), ADX(14) via `pandas_ta` (`prepare_data` in `backtest_20d_realworld.py`).
   - PnL calculations: Net PnL is computed per trade as `pos['size'] * pnl_pct` where `pnl_pct` subtracts `fee_round_trip` (0.0008 = 0.08%) in `core/replay_engine.py:43-45`.
   - Fees & Slippage: Slippage is set to `bot.REPLAY_SLIPPAGE_PCT = 0.0002` (0.02% per side, applied to both entry fill in line 152 and exit price in line 42).
   - Position Compounding: Sizing dynamically compounds based on current account equity `balance` (`ideal = balance * params['risk_pct'] / max(stop_pct, 0.001)`, `size = min(ideal, balance * cap_per_trade * leverage, available * leverage, 10000.0)` in `core/replay_engine.py:156-158`).

2. **`scripts/parity_check_24h.py`**:
   - Both simulated live paper mode (`run_live_engine`) and parity comparisons run on `core/replay_engine.py:run_live_replay`, ensuring parity check compares authentic paper execution logic against historical data.
